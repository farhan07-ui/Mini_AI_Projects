# 🤖 AI Mini Projects

A collection of AI-powered mini applications built using **Python**, **Flask**, and **Large Language Models (LLMs)**. This repository demonstrates practical AI use cases including text generation, document analysis, image understanding, code assistance, and intelligent chatbots.

---

## 📂 Projects Included

### 1. AI Image Caption Generator
Generate captions for uploaded images using AI.

### 2. AI Meeting Notes Generator
Convert meeting discussions into concise summaries and action items.

### 3. AI Email Generator
Generate professional emails and rewrite emails in different tones.

### 4. AI Code Assistant
Explain code, detect bugs, generate functions, and convert code between programming languages.

### 5. AI Resume Analyzer
Analyze resumes and provide AI-powered feedback and improvement suggestions.

### 6. AI Recipe Generator
Generate recipes based on available ingredients.

### 7. AI Research Assistant
Answer research queries and summarize information using AI.

### 8. AI Personal Finance Assistant
Provide budgeting tips, savings suggestions, and financial guidance.

### 9. AI Medical Chatbot
Provide general health-related information and answer medical queries for educational purposes.

### 10. AI Legal Document Assistant
Summarize legal documents and explain legal terms in simple language.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/farhan07-ui/Mini_AI_Projects.git
```

Go to any project folder:

```bash
cd 03_ai_email_generator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your API key.

Run the application:

```bash
python app.py
```
